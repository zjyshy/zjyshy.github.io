1，基本概览

1.1 常见符号

~ 用户目录

/ 代表硬盘

. 当前目录

$ 可以输入命令

.. 上级目录 比如有一个目录里面放了index和img两个目录，index里面放了index.html文件，img里面有一张图叫hehe.jpg，如果index.html里面想要访问这张图.

../img/hehe.jpg<br>

  • 相对路径

2，自带命令

2.1 目录

  • cd 进入目录

    进入当前用户目录

    cd ~

    返回上级目录

    cd - 
cd ..

    如果知道绝对路径,就可以像下面一样去访问

    /Users/apple/hexoStorage/public
  • pwd 显示当前目录的绝对路径
  • mkdir

    创建一个目录

    mkdir test

    创建一个嵌套的目录,加参数p，这里的引号可有可无

    mkdir -p "test-1/test-2/test-3/test-4"

    创建多个连续的目录,如下命令创建test1-test5这五个目录

    mkdir test{1..5}

    创建多个，不连续的目录

    mkdir test index

    创建名字中含有空格的目录,这里的引号是必须的，如果没有引号就是会导致创建两个目录test 和 1。

    mkdir "test 1"
  • whoani 查看当前用户目录名

2.2 路径

主要就是一个ls命令，以及它的相关参数

列出当前路径的所有非.内容

ls

列出当前路径下某个路径的内容

ls directory

列出当前路径下所有内容（包含以.开头的） .开头的隐藏文件

ls -a

列出更多信息 关于-l参数显示的内容

ls -l

列出包括隐藏文件在内的更多信息

ls -la

2.3 创建文件

  • echo

    创建文件,将文本hello world 存储到文本文件test1.txt，如果test1.txt不存在就会新建一个test1.txt并把文本内容写进去；如果已经存在一个非空的test1.txt那就会将原有内容覆盖。

    echo "hello world" > test1.txt

    如果想在已有的内容后面追加内容

    echo "heihei" >> test1.txt
  • touch

    touch在创建的时候并不可以为文件添加内容，可以改变文件最后的更新时间（感觉没啥用）。

    touch test1.txt

    2.4 复制

  • cp

    复制文件，将文件1的内容复制到文件二。如果是一个未创建的文件那就会创建一个文件，并将内容复制进去。如果是已经创建的文件，就会覆盖掉原有内容。

    cp test1.txt test2.txt

    复制目录，如果是复制到一个不存在的目录，就会新建此目录；如果是复制到一个已存在的目录，则会覆盖掉原有目录；如果目录里面有文件,则会将文件复制过去

    cp -r test1 test2

2.5 移动和重命名

  • mv

    mv并不会自动新建不存在的目录。

    mv 文件名 文件名       将源文件名改为目标文件名

    mv 文件名 目录名        将文件移动到目标目录

    mv 目录名 目录名        目标目录已存在，将源目录移动到目标目录；

                                         目标目录不存在则改名

    mv 目录名 文件名        出错

2.6 删除

  • rm

    参数

    -i 删除前逐一询问确认。

    -f 即使原档案属性设为唯读，亦直接删除，无需逐一确认。

    -r 将目录及以下之档案亦逐一删除。
