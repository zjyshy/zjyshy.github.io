2，为什么是HTTP

我想说一下，其实Http知识Tcp/Ip协议的一个子集，还有诸如FTP、IP、DNS等等协议。但是为什么我们主要学习HTTP就ok了呢，因为TCP/IP是分层的：

  • 应用层

  • 传输层

  • 网络层

  • 网络接口层

    而这个Http就是处于离我们最近的应用层，其他的层次以及协议也一样重要，但是我们并不用深入的去学，因为对于前端开发的帮助不大。并且在应用层也并不是只有Http一家独大，但是我们主要用的就是http。
