一，不同符号的意思

~

用户目录：用户名所命名的目录。 

cd ~  \\可以打开用户目录（当前用户）

/

在Linux和Unix硬盘用’/‘来表示，还可以表示分界线

/Users/apple  \\这里第一个/表示整块硬盘第二个表示分界线

$

表示可以输入命令

二，常见命令

cd （change directory）

mkdir （make directory）

rm （remove）

touch

mv （move）
